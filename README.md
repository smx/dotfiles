# Installation
### Download and change folder
```
cd ~
git clone https://github.com/smx/dotfiles.git

cd dotfiles
```
### Run install
```
sudo chmod a+x install.sh
```
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
